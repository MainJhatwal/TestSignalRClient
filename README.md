This is a sample client designed to test SignalR hubs. To set up the code for usage, simply follow the steps below:

Setup Instructions
Open the program.cs file.
Locate the baseURL variable and set it to the URL of your SignalR hub.
Press F5 or run the client to initiate testing.
Remember to have the necessary dependencies installed and ensure that you have a functional SignalR hub to connect to.

Feel free to customize the client according to your testing requirements and preferences. Happy testing!
